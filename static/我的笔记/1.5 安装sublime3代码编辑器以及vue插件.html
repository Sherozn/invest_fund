从第二章开始我们就要正式开始敲代码了，微信开发者工具里面的代码编辑器不太好用，我们需要自己安装一个sublime代码编辑器。

1、下载安装包
点击下面链接，根据自己的电脑版本下载对应的安装包
http://www.sublimetext.com/3

Mac电脑就下载第一个OS X版本的安装包就可以，下载的安装包名称为 Sublime Text Build 3207.dmg，直接双击在界面安装打开即可，打开sublime界面如下：


2、安装vue插件
command+shift+p，输入install，选择第一个
这里可能会出现错误，可以看本文末尾有错误解析，如果没有包含你的错误，可以留言哦~

需要等一会，出现一个输入框，搜索Vue，点击Vue Syntax Highlight插件安装，安装需要几分钟的时间

安装完成后，点击右下角Plain Text出现Vue Component选择就说明安装成功了


3、可能出现的错误

（1）错误出现原因：
点击菜单栏Sublime Text->Preferences->Package Setting->Package Control ->Setting Default打开文件
文件中有几行这样的代码：
"channels": [
  "https://packagecontrol.io/channel_v3.json"
],
就是他们搞的鬼，意思是channel_v3.json文件是从网络连接获取的，目前这个https链接请求超时。所以我们的解决方法是将这个文件下载下来放在本地，然后直接从本地获取这个文件。
"channels": [
  "https://packagecontrol.io/channel_v3.json"
],

（2）问题解决
点击下面链接下载channel_v3.json文件，放在你电脑上的一个文件夹里
https://pan.baidu.com/s/1uTMD2PECrE5ItKaUmDy4Qg    
提取码：f756

然后打开Preferences->Package Setting->Package Control ->Setting User （这里需要注意的是Setting User，不是我们刚刚打开的Setting Default）设置中添加下列代码，问题就解决了。
其中`"/Users/xuzhaoning/workplace/channel_v3.json"`是我电脑上channel_v3.json文件的路径，需要改成你自己电脑上面的文件路径~
"channels":
[
  "/Users/xuzhaoning/workplace/channel_v3.json"
],
可以对比下面的照片，检查一下自己代码添加的位置是否正确。添加完成后，按command+s（window电脑是ctrl+s）保存文件，重新打开sublime再次操作就没有问题了~
